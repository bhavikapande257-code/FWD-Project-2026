# Personal expense tracker; A minimalist web tool designed  to empower users to take control of their finances through real-time expense logging and visual spending breakdowns.

Live demo(for viewing live site here)
Features:
1) Dynamic entry; Add expenses with a title amount and date
2) Categarization tabs for food , transport , entertainment etc

Tech Stack: *HTML: Semantic structure for financial data. 
             CSS:Layout using Flexbox/Grid
             JavaScript: DOM manipulation for adding/removing list items.
